This repository contains a comprehensive IPL Data Analysis project developed using Power BI. The project analyzes historical Indian Premier League (IPL) data to uncover key performance insights, team dynamics, and match-winning factors.

📊 Project Overview
The primary goal of this project is to leverage sports analytics to identify the most successful teams and players, understand the impact of playing conditions, and analyze toss decisions over various IPL seasons.
Key ObjectivesPerformance Analysis: Identify top-performing batsmen (highest scorers) and bowlers (most wickets).Strategic Insights: Analyze toss trends and their influence on match outcomes.Venue & City Trends: Identify hosting patterns and "lucky" stadiums for top teams.Interactive Exploration: Provide a user-friendly dashboard with filters for seasons, teams, and players.

Tools & SkillsTool: Power BI Skills: * Data Cleaning & Transformation (Power Query) Exploratory Data Analysis (EDA) DAX for complex measures and insights Data Visualization & Dashboard Design 

📂 Dataset Description
The analysis is based on two primary CSV datasets:


Matches Dataset: Contains match-level details like season, city, toss winner, match winner, and venue.


Deliveries Dataset: Contains ball-by-ball information, including batsman runs, extra runs, and dismissal types
